# Conversor de Moedas (Java)

Aplicação desenvolvida em Java com interface gráfica (Swing) para conversão de valores em Real para Dólar e Euro.

## Funcionalidades
- Conversão de Real para Dólar
- Conversão de Real para Euro
- Interface gráfica com múltiplas telas

## Tecnologias
- Java
- Swing (Interface gráfica)

## Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Eventos e interface gráfica
- Organização em classes

## Objetivo
Projeto desenvolvido para prática de desenvolvimento de aplicações com interface gráfica em Java.
